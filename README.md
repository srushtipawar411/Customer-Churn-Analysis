# Customer Churn Analysis Dashboard

##  Project Overview
Customer Churn Analysis is a Power BI dashboard designed to identify customer attrition patterns and the key factors influencing churn. The dashboard provides interactive visualizations that help businesses understand customer behavior, improve retention strategies, and make data-driven decisions.


##  Objectives
- Analyze customer churn trends.
- Identify high-risk customer segments.
- Compare churn across demographics and services.
- Understand the impact of contracts, payment methods, and monthly charges.
- Provide actionable insights to improve customer retention.


##  Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)


##  Dataset
The dataset contains customer information including:
- Customer demographics
- Contract type
- Internet services
- Payment methods
- Monthly & total charges
- Tenure
- Churn status


##  Dashboard Features
- Overall customer and churn summary
- Churn rate analysis
- Customer segmentation
- Churn by gender and senior citizens
- Churn by contract type
- Churn by payment method
- Churn by internet service
- Monthly charges and tenure analysis
- Interactive filters and slicers


##  Key KPIs
- Total Customers
- Total Churned Customers
- Churn Rate
- Monthly Revenue
- Average Monthly Charges
- Customer Tenure


##  Business Insights
- Identify customer groups with the highest churn probability.
- Analyze how contract types affect customer retention.
- Understand the relationship between tenure and churn.
- Discover payment methods associated with higher churn.
- Support data-driven customer retention strategies.
